# AGENTS.md

This repository contains an SSD firmware fuzzer for reliability and robustness testing on owned or authorized test devices.

The project goal is to find crashes, hangs, undefined behavior, timeout issues, data corruption, and other stability problems. It is not intended for exploitation, unauthorized access, persistence, evasion, or attacking third-party systems.

When assisting with this repository, focus on safe code review, bug finding, reliability improvements, defensive programming, refactoring, and test suggestions.

For code reviews, check for logic errors, boundary issues, race conditions, memory/resource leaks, unsafe assumptions, error handling gaps, reproducibility problems, and missing tests.

Do not add exploit logic, stealth behavior, unauthorized access logic, credential handling, or offensive attack instructions.

For code review responses, use:

1. Critical issues
2. Potential bugs
3. Reliability improvements
4. Suggested tests
